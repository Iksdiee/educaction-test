Веб-технологии

Ведение в веб-технологии

Веб-технологии - это технологии, которые используются в процессе коммуникации между различными девайсами при работе с интернет сетью, например - браузер.

Браузер используется всеми для отображения веб-страниц. Простыми словами, браузер - это программа, используемая для воспроизведения текста, информации, видео, аудио и анимаций из интернета. Однако, под казалось бы простым описанием браузеров скрывается очень много сложных, но в то же самое время очень интересных технологий, которые позволяют любому человеку пользоваться интернетом, обмениваться информацией, выставлять фотографии в социальных сетях и поддерживать контакт с дальними родственниками при помощи электронной почты.

В свою очередь компьютеры, для общения между собой, не используют привычный для человека язык, например русский или английский языки. При работе с интернетом, "общение" происходит посредством обмена пакетами через всемирную сеть между двумя независимыми друг от друга компьютерами. Одним из языков такого общения между компьютерами является язык HTML, о котором я расскажу подробнее.

Что такое HTML, и почему этот язык программирования является основопологающим

HTML (от английского HyperText Markup Language) — это язык гипертекстовой разметки страницы. Он используется для того, чтобы дать браузеру понять, как нужно отображать загруженный сайт.

Язык HTML был разработан Тимом Бернерс-Ли во время его работы в CERN и распространен браузером Mosaic, разработанным в NCSA. В 1990-х годах он добился особенных успехов благодаря быстрому росту Web. В это время HTML был расширен и дополнен. В Web очень важно использование одних и тех же соглашений HTML авторами Web-страниц и производителями. Это явилось причиной совместной работы над спецификациями языка HTML. HTML 2.0 (ноябрь 1995) был разработан под эгидой Internet Engineering Task Force (IETF) для упорядочения общепринятых положений в конце 1994 года. HTML+ (1993) и HTML 3.0 (1995) - это более богатые версии языка HTML. Несмотря на то, что в обычных дискуссиях согласие никогда не было достигнуто, эти черновики привели к принятию ряда новых свойств. Усилия Рабочей группы World Wide Web Consortium по HTML в упорядочении общепринятых положений в 1996 привели к версии HTML 3.2. Ныняшняя версия языка - HTML5 была разработана в 2017 году и на сегоднящний день все современные сайты используют именно эту версию языка.

Как работает HTML?

Язык состоит из тэгов-своеобразных команд, которые преобразуются в визуальные объекты в браузере пользователя.

Принцип работы разметки следующий:

  1. Вы вводите в адресную строку адрес сайта или страницы.
  2. Браузер делает запрос по этому адресу и получает файл в формате.html.
  3. Код из полученного файла преобразуется в визуальные объекты.

Например, тег <img> используется для размещения изображений на странице. У него есть обязательный атрибут src, в котором указывается ссылка на файл. При помощи данного тега можно, например, добавлять на страницы изображения собак.

Возможности HTML

На вставлении картинок, возможности данного языка не заканчиваются. При помощи HTML, есть возможность создавать различные списки и отображать их на странице. Нумерованные списки будут полезны для создания списков, в которых важна последовательность, пример такого списка можно было увидеть ранее, когда описывался принцип работы разметки. Помимо нумерованных списков так же есть возможность создания маркированых списков. Такие списки будут удобны, например, для создания списка покупок в магазин.

  • Помидоры
  • Огурцы
  • Молоко
  • Хлеб
  • Сыр

Помимо списков, используя HTML, можно создавать таблицы, которые могут пригодиться в разных сферах. Например, для создания списка товаров, с указанием их стоимости. Для создания таблицы, используется тег <table>. Для обозначения столбцов и рядов в таблице используются теги <td> и <tr> соответственно.

Область применения HTML

HTML - это язык разметки страницы, при помощи него формируется именно каркасс сайта, что и где будет отображаться, расположение элементов страницы относительно друг друга как раз определяется данным языком. Однако, в современных реалиях, знания только HTML не достаточно для работы в сфере веб-программирования. Как упоминалось ранее, данный язык применяется только для разметки страницы. Для стилизации страницы используется язык CSS при помощи которого, визуальная составляющая страницы начинает принимать форму, приятную для глаза и удобную для воприятия инофрмации на сайте. Подробнее о данном языке можно почтитать здесь. Для придания сайтам интерактивности, используется другой язык - JavaScript, подробнее о данном языке можно узнать на википедии или здесь.
